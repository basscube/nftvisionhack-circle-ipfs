# NFT Vision Hackthon IPFS&Circle <> Basscube

## Dev
- Install
    ```
    yarn
    ```
- Run node
    ```
    yarn run staging:node
    ```
- Delpoy contract(clean config.json first)
    ```
    yarn run staging:deploy
    ```
- Run local front-end
    ```
    yarn start
    ```
- Change metamask RPC

Circle Payment related in /payment folder
